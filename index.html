Submit/Reset Buttons

CSS selector:

.button_confirmation

Sample code:


          <button class="button_confirmation" type="reset"> Clear </button> 
          <button class="button_confirmation" type="submit"> Submit </button>

Text Input Field with Label

CSS selector:

.form_general

Sample code:


          <form class="form_general" name="feedback_form">
            <label for="comment">Comments: </label><input type="text" name="comment" id="comment">
          </form>

Rendered element:

Radio Button with Label

CSS selector:

.form_radio

Sample code:


          <form>
            <input type="radio" name="quantity_option" value="quantity1" id="quantity1"> <label for="quantity1">1</label>
          </form>

Rendered element:
